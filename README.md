# even-filter

Pythonの条件分岐（if文）とループ（while / for）の練習用リポジトリです。
## 学習内容

- while文・for文を使った繰り返し処理
- if文による条件分岐
- and / or / not を使った複数条件の組み合わせ
- 条件を変数に分解して可読性を上げる書き方
## 例

0〜50の数字の中から  
「偶数」「10以上」「4の倍数ではない」  
数を抽出する処理を実装しました。
